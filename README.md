## Description
awesome is ecom opensource platform, generated using QOR5 framework.

## How To Install
```bash
git clone https://github.com/Shurik12/awesome.git
cd awesome
docker-compose up -d
source dev_env
go run main.go 
```

## Create binary in project folder
```bash
go mod init awesome
go get
go build
```

## Create binary in go folder (GOPATH/bin/)
```bash
go install
```
